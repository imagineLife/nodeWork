# Evaluating Code

- checking code
- using a node/js file across platforms, check syntax of a program that is directly as a shell script!

## Eval to evaluate

```bash
node --eval "2 * 4"
```

(no output)

```bash
node -e "console.log(14+7)"
```

```bash
21
```

## Print to evaluate and print

```bash
node --print "4 * 5"
```

`20`
